 - [x] **Alpine State for Theme**: Initialize Alpine in `base.html`:
   ```html
   <html x-data="{ theme: localStorage.getItem('theme') || 'light' }" data-theme="theme">
   ```

   ```html
   <input type="checkbox" class="toggle"

   ```
